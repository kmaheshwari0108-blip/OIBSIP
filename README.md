# OIBSIP

Exploratory Data Analysis (EDA) on Retail Sales Data

**Project Overview**

This project performs Exploratory Data Analysis (EDA) on retail sales data to identify sales trends, customer purchasing behavior, and product performance.

**Objectives**

- Clean and preprocess retail sales data
- Calculate descriptive statistics
- Analyze sales trends over time
- Study customer demographics and purchasing patterns
- Visualize insights using charts and heatmaps
- Provide business recommendations

**Tools & Technologies**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

**Key Insights**
- Identified top-performing product categories
- Analyzed monthly sales trends
- Examined customer demographics
- Discovered relationships between sales variables

**Business Recommendations**
- Focus marketing on high-spending customer segments
- Maintain inventory for top-selling products
- Run promotions during low-sales periods

**Project Structure**
- Data Loading & Cleaning
- Descriptive Statistics
- Customer Analysis
- Product Analysis
- Time Series Analysis
- Data Visualization
