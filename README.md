# OIBSIP
Data Analytics Internship Projects

This repository contains projects completed during my internship, focusing on Data Analysis using Python.

The projects demonstrate practical applications of Exploratory Data Analysis (EDA) and Predictive Modeling to solve real-world business problems.

---

**Projects Included**

**Task 1: Retail Store Sales Analysis (EDA)**

**Project Overview**

This project focuses on performing Exploratory Data Analysis (EDA) on retail sales data to uncover trends, customer behavior, and product performance.

**Objectives**

- Load and clean retail sales data
- Perform descriptive statistical analysis
- Analyze sales trends over time
- Study customer demographics and purchasing behavior
- Visualize insights using charts and heatmaps
- Generate business recommendations

**Key Techniques Used**

- Data Cleaning
- Descriptive Statistics
- Time Series Analysis
- Customer Analysis
- Product Analysis
- Data Visualization

**Technologies Used**

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

**Key Insights**

- Identified top-performing product categories
- Analyzed monthly sales trends
- Studied customer purchasing patterns
- Generated actionable business recommendations

---

**Task 2: House Price Prediction using Linear Regression**

**Project Overview**

This project develops a Machine Learning model to predict house prices based on housing features such as area, bedrooms, bathrooms, and parking availability.

**Objectives**

- Explore and preprocess housing data
- Select relevant features
- Train a Linear Regression model
- Predict house prices
- Evaluate model performance
- Visualize prediction results

**Key Techniques Used**

- Data Preprocessing
- Feature Selection
- Linear Regression
- Model Evaluation
- Data Visualization

**Technologies Used**

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

**Evaluation Metrics**

- Mean Squared Error (MSE)
- R-Squared (R² Score)

**Results**

The model successfully predicts house prices and demonstrates the practical implementation of supervised machine learning technique
