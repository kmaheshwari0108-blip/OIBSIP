**Task 1: Retail Store Sales Analysis (EDA)**

**Project Overview**

This project focuses on performing Exploratory Data Analysis (EDA) on retail sales data to uncover trends, customer behavior, and product performance.

**Objectives**

- Load and clean retail sales data
- Perform descriptive statistical analysis
- Analyze sales trends over time
- Study customer demographics and purchasing behavior
- Visualize insights using charts and heatmaps
- Generate business recommendations

**Key Techniques Used**

- Data Cleaning
- Descriptive Statistics
- Time Series Analysis
- Customer Analysis
- Product Analysis
- Data Visualization

**Technologies Used**

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

**Key Insights**

- Identified top-performing product categories
- Analyzed monthly sales trends
- Studied customer purchasing patterns
- Generated actionable business recommendations
